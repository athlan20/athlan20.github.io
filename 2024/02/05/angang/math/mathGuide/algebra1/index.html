    [02-2]一般与特殊

    交换律

    1+2=2+1 是特殊的等式,m+n=n+m就是一般的等式
    2*3=3*2 是特殊的等式,m*n = n*m就是一般的等式

    结合律

    45+28+72 = 45+(28+72) 是特殊的等式, a+b+c=a+(b+c)就是一般的等式
    6*25*4 = 6*(25*4) 是特殊的等式,a*b*c = a*(b*c)就是一般的等式

    分配率

    5×(2+4)=5×2+5×4 是特殊的等式,a×(b+c)=a×b+a×c就是一般的等式

    [02-3]一般与特殊的相对性

    奇数和3相比,奇数是一般,3是特殊,但是奇数和自然数相比,自然数是一般,奇数是特殊

    概念的分类

    分类的要求-不漏不重

    把一个一般概念分成它的特殊概念（比如把服装分成男装和女装）

    1. 选定一个分类标准，进行分类。
    2. 不许遗漏
    3. 不许重复

    二分法

    第一类通常指特定对象（狗）
    第二类是指其他的对象（非狗）

    命题

    能够判断真假的语句统称命题

    p或q

    只有当p,q两个都为假，才为假

    p且q

    只有当p,q两个都是真时，才为真

    p和非p（矛盾命题）

    必定是一真一假

    若p则q（条件命题）

    从条件（p）得出结论（q）则为真，否则为假

    1. 原命题 若p则q
    2. 逆命题 若q则p
    3. 否命题 若非p则非q
    4. 逆否命题 若非q则非p
      1和4,2和3互为逆否，互为逆否的两个命题同真同假（等价）
      如果能从“非q”推出“非p”，就表明“若p则q”是真命题

    演绎推理

    [[#找共性]] 就是特殊到一般(一般性就是共性)
    [[#用共性]] 就是一般到特殊(演绎推理)

    三段论

    1. 末两位是25的整数都能被25整除
    2. 因为732425末两位是25.
    3. 所以732425能被25整除
      第一句话是”一般”,也叫”大前提”
      第二句话是”特殊”,也叫”小前提”
      第三句话是”结论”
      一个演绎推理由”一般”,”特殊”,”结论”三个部分组成

    省略三段论

    当大前提和小前提非常熟悉时,大小前提都可以省略,直接说结论
