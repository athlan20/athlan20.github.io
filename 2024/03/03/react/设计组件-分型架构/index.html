一个tsx同时可以被当成应用，又能被当做组件使用，称为「分形架构」

就是我们拿来需要分形的tsx,最终包装成IconPicker提供出去

  1. useStoreUpdater ：将外部的 props 同步到 store 内部的方法
  2. StoreUpdaterProps：从 store 的 State 中 pick 出需要受控的状态，并相应补充 defaultXX 的 props
  3. StoreUpdater：逐一补充调用外部组件 props 的受控状态，将外部 props 更新到 store 的内部状态中；
