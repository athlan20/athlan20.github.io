  1. 标题 (Headings)

    • # 一级标题 会显示为：

      一级标题

    • ## 二级标题 会显示为：

      二级标题

    • ### 三级标题 会显示为：

      三级标题

    • 以此类推，直到六级标题。

  2. 粗体 (Bold)

    • **粗体文本** 或 __粗体文本__ 会显示为： 粗体文本

  3. 斜体 (Italic)

    • *斜体文本* 或 _斜体文本_ 会显示为： 斜体文本

  4. 链接 (Links) ^049eed

    • [链接文本](http://url.com) 会显示为： 链接文本

  5. 无序列表 (Unordered List)

    • diffCopy code

      - 列表项1 - 列表项2 - 列表项3

      会显示为：

      • 列表项1
      • 列表项2
      • 列表项3

  6. 有序列表 (Ordered List)

    • markdownCopy code

      1. 第一项 2. 第二项 3. 第三项

      会显示为：

      1. 第一项
      2. 第二项
      3. 第三项

  7. 图片 (Images)

    • ![替代文本](http://url-to-image.com) 会在Markdown支持的平台上显示为图片。

  8. 引用 (Blockquote)

    • > 引用文本 会显示为：

      引用文本

  9. 代码 (Code)

    • 行内代码 使用单个反引号：code 会显示为 code

    • Copy code

      代码块

      使用三个反引号或四个空格缩进会显示为代码块。

  10. 水平线 (Horizontal Rule)

    • --- 或 *** 会显示为一条水平线。

这些是Markdown中最常用的格式。
